This code can be used to visualize the trips of a team in a TTP distance.
To use it, run:
python3 viz.py --instance instance.xml --solution sol.xml --output output.pdf

Or to test it with default values:
python3 viz.py

In order to translate the distance matrix into coordinates, we make use of MDS.
This means that the coordinates are merely an estimate. MDS especially has difficulties 
correctly translating the LINE instances. For the constant distance instances, note that
no correct 2D plot exists.
